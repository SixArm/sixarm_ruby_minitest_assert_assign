Introduction

Minitest extensions for our common cases:

Assert:

  • assert_true

  • assert_false

  • assert_respond_to_all

  • assert_equal_items

Spec:

  • must_be_true

  • must_be_false

  • must_respond_to_all

  • must_have_equal_items

Based on Minitest.

For docs go to sixarm.com/sixarm_ruby_minitest_assert_assign/doc

Want to help? We're happy to get pull requests.

Install

Gem

Run this command in your shell or terminal:

gem install sixarm_ruby_minitest_assert_assign

Or add this to your Gemfile:

gem 'sixarm_ruby_minitest_assert_assign'

Require

To require the gem in your code:

require 'sixarm_ruby_minitest_assert_assign'

Examples

Example Minitest spec:

describe Foo do
  it "validates" do
    f = Foo.new
    f.valid?.must_be_true
  end
end

Example Minitest spec:

describe Foo do
  it "is colorful" do
    f = Foo.new
    f.must_respond_to_all [:red, :green, :blue]
  end
end
